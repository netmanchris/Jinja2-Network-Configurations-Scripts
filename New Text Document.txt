#Jinja2 Configuration Scripts

This repository includes jinja2 files which are intended to help automate the creation of network infrastructure device configurations. 

Many of the scripts have been created using the HP CLI guide as a reference source. It is freely available with registration at the [HP Press](https://h30590.www3.hp.com/product/HP+Networking+and+Cisco+CLI+Reference+Guide+-+Version+3-PDF-15901) website as an ebook. 

These scripts are provided as is and it is HIGHLY recommended that a network professional examine all of the resulting configurations before attempting to use them in a production environment.
